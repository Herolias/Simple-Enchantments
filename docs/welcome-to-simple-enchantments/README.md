Simple Enchantments adds a complete enchanting system to the game. Currently, the mod includes an Enchanting Table, an Engraving Table, <!-- DOCSTAT:enchantments.count -->33<!-- /DOCSTAT --> unique enchantments, a detailed config to customize almost every part of the mod, 3 commands, and an API for developers to create their own enchantments or register custom weapons.

This documentation is split into 3 parts:
* **Getting Started:** For general gameplay and getting started.
* **[Server Owner Guide](https://wiki.hytalemodding.dev/mod/simple-enchantments/server-owner-guide):** A breakdown of commands and config files.
* **[Enchantment API Documentation](https://wiki.hytalemodding.dev/mod/simple-enchantments/api-documentation):** For developers looking to build on top of the mod.

## History

On my first playthrough of Hytale, I really missed the enchantment system from Minecraft. I wanted my tools and weapons to feel more personal and grow stronger as I progressed through the game. That’s when I came up with the idea for Simple Enchantments, and after talking it over with Sorath, we decided to make it a reality.

We didn't want to just copy Minecraft’s system, though. It has its own flaws and Hytale is a very different game. While some of our enchantments are inspired by Minecraft, many are unique and designed specifically to fit the way Hytale plays.
