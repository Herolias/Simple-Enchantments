# The Enchanting Table

![](https://raw.githubusercontent.com/Herolias/Simple-Enchantments/main/docs/media/images/A5C9l1w-70df1f50b9.png)

To get started with Simple Enchantments, you need to craft an Enchanting Table at the Workbench. 

![](https://raw.githubusercontent.com/Herolias/Simple-Enchantments/main/docs/media/images/69v3FeX-961b221dc1.png)

The Enchantment Table is used to craft [Scrolls](scrolls), which are used to apply enchantments to your Items.
The Table has <!-- DOCSTAT:enchantingTable.tier.count -->4<!-- /DOCSTAT --> Upgrade Tiers, and each Tier unlocks stronger enchantments, with the <!-- DOCSTAT:enchantingTable.tier.count;ordinal -->4th<!-- /DOCSTAT --> Tier unlocking special legendary enchantments.
