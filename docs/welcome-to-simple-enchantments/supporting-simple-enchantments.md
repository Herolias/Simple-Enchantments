# Supporting Simple Enchantments
If you would like to support the development of Simple Enchantments, there are many ways to contribute:

- Help improve the translations in your native language. You can make a [PR on GitHub](https://github.com/Herolias/Simple-Enchantments) or [DM me](https://discord.com/users/herolias) about it.
- Submit detailed bug reports or feature requests.
- Give the project a star on [GitHub](https://github.com/Herolias/Simple-Enchantments). It costs nothing and makes us happy.
- Join the Discord and talk with us about how we can improve the mod.
- Use code `Herolias` when purchasing or upgrading your Hytale Edition.

If you plan on hosting a Hytale server, consider using Bisect Hosting with our code. It gives you a discount and supports us with a commission. We genuinely think Bisect is a good hosting provider, and their integrated mod manager makes it easy to download and update Hytale mods.
