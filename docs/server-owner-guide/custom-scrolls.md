# Custom Scrolls

If you want to reward your players with a combination of enchantments or an OP enchantment higher than the normal max level of that enchantment, you can do so with the custom scroll. The blank custom scroll is a creative inventory only Item that can be enchanted via Enchantment Scrolls or via the /enchant command (or /give -enchant) with enchantments up to level 100. Players with these scrolls can then apply these enchantments one after another to the same or different Items. If the last enchantment is applied, the scroll vanishes.
