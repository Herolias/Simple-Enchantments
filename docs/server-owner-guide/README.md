Welcome to the Simple Enchantments documentation for server owners. This guide focuses on customizing the mod to fit your server. Simple Enchantments is built from the ground up with server customization in mind, so you can adjust nearly every part of the mod: scroll recipes, table recipes, table upgrade costs, crafting availability, individual enchantments, enchantment strength, and more. If a customization option still feels missing, join the Discord server or leave a comment on the mod page.
