# Server-Owner Guide

Welcome to the Simple Enchantments Documentation for Server Owners. This Documentation focuses on how you can customize the mod to fit your Server. Simple Enchantments is build from the ground up with Server Customization in mind and you can basically customize every part of the mod: Scroll and Table Recipies, disabling crafting, disabling individual enchantments or tweaking the strength of the Enchantments and much more. If you still feel like a customization option is missing, join the Discord Server or write a comment on the mod page.
