# The Enchantments Page

On the Enchantments Page, you can enable/disable individual enchantments as well as changing their modifier(s) if they have one.
The modifier value is in percent (0.10 = 10%) and per level, and for Enchantments that goes on multiple Armor slots (like protection) it is also per Armor piece.
