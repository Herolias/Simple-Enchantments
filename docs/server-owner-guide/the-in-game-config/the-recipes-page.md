# The Recipes Page

On this page you can edit the crafting recipies of every enchantment scroll and set at which level they unlock at the Enchanting Table. If you click on the Edit Recipe Button you get to the Recipe Editor. In the recipe editor, you will see the standard recipe of that scroll, as well as buttons to adjust the quantity of the ingredients, change their order or add/remove/change ingredients. If you add/change an ingredient, a search bar on the right side will pop up, letting you search for specific Items. The searchbar contains every Item in the game (including creative only and mod Items), so make sure to check if your preffered Item is optainable in survival. You can in theory add as many different ingredients as you want, exept for the Table Upgrade Costs. For these, there is a maximum of 5 ingredients, more then that is not supported by the Hytale UI yet (will still work, but looks weird).
