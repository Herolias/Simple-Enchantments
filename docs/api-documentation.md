# API Documentation
The documentation for the Simple Enchantments API can be found on [GitHub](https://github.com/Herolias/Enchantment-API-Example). We will also create detailed documentation for it here in the wiki soon. The API lets other modders and server owners create their own enchantments or read information about enchantments on specific items.
